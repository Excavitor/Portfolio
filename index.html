Publication

Identifying Threats on Social Media to Spot Offensive Behavior | NLP Project

Successfully developed and implemented classification models to detect offensive content in Bangla text from social media platforms. We analyzed 44,000 social media comments to detect offensive text using five transformer models. XLM-RoBERTa-base achieved the highest accuracy (83.54%), while Multilingual BERT had the highest AUC (0.85). Variations in learning rate, dropout rate, epochs, early stopping, and batch size were tested to optimize performance.
